== 📦 Instalacja

[source,bash]
----
git clone https://github.com/twoj-login/twoj-projekt.git # <1>
cd twoj-projekt                                          # <2>
npm install                                              # <3>
----

<1> Klonuje repozytorium
<2> Przechodzi do katalogu projektu
<3> Instaluje zależności

== ▶️ Uruchomienie
